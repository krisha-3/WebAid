# WebAid - Website Accessibility Interface Design

WebAid is a browser-based accessibility tool that enhances the readability and usability of any website in real time. It works on the user side, requiring no modifications from website owners.

The extension intelligently detects the main content of a webpage and applies adaptive styling such as improved text spacing, contrast, and layout. It is designed to provide a more comfortable and inclusive browsing experience, especially for users with visual and cognitive challenges.


## Features

- Intelligent content detection for focused readability
- Improves text clarity, spacing, and layout
- Customizable accessibility settings
- Works on any website without integration
- Lightweight with minimal performance impact
- Fully reversible changes (reset anytime)
- Consistent experience across different websites


## Tech Stack

- JavaScript
- CSS
- Browser Extension APIs


## Used By

This project is used by:

- Students who read long articles and study online
- Working professionals handling dense content daily
- Senior citizens who need better readability
- Users with dyslexia, ADHD, or other reading difficulties
